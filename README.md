Antikorper
==========

Antikorper is a WIDS which captures the intraWLAN traffic and the 802.11 raw traffic. In live it process thist traffic and detect anomalies and intruders inside or around the network to protect.
Among the modules which composes Antikorper, we can find:

-MAC Filtering
	To prevent intruders inside the network

-ARP Traffic processing
	To prevent ARP spoofing, poisoning... and the infinity of attacks arising from them.

-802.11 raw Traffic processing 
	To detect Rogue AP's around the network to protect.

-Performer
	The Antikorper performer allows the system to act against a threat, like a not authorized machine or a Rogue AP around the network. This will be automatically done by the system by the 		dissasociation of the intruder from the network.
